Sistema de Gestión Académica (SGA)

Proyecto Final - Módulo 3: Programación en Python

:pencil: Descripción Este sistema es una solución robusta y modular diseñada para automatizar la administración de datos académicos en una empresa tecnológica. Permite gestionar el ciclo de vida completo del estudiante: desde el registro inicial y la inscripción de materias, hasta el cálculo automatizado de promedios mediante lógica recursiva.

:tools: Stack Tecnológico Para este desarrollo se aplicaron conceptos avanzados de Python:

Programación Modular: División lógica en main.py, estudiantes.py y validaciones.py.

Estructuras de Datos Eficientes:

Dictionaries para perfiles de alumnos.

Sets para garantizar materias únicas.

Lists para historial de notas.

Recursividad: Cálculo de promedios optimizado.

Persistencia: Manejo de archivos .txt para reportes finales.

:open_file_folder: Estructura del Proyecto

├── src/ │ ├── main.py # Interfaz de usuario y menú principal │ ├── estudiantes.py # Lógica de negocio y base de datos │ └── validaciones.py # Control de errores y excepciones

├── docs/ │ ├── Documentacion_tecnica.pdf │ ├── informe_validacion.pdf │ └── Presentación.pdf └── datos_estudiantes.txt # Reporte generado automáticamente

:gear: Instalación y Uso Clona el repositorio:

git clone https://github.com/tu-usuario/Sistema-Gestion-Academica.git

Ejecuta el programa principal:

python src/main.py
